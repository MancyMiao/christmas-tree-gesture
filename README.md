# 手势交互圣诞树 🎄

这是一个基于 **Three.js** 和 **MediaPipe** 实现的 3D 圣诞树交互项目。通过手势识别（例如张开手、握拳、捏合等），用户可以与圣诞树进行交互，触发不同的粒子效果、挂上照片以及更多功能。

## 功能特性

- **手势识别**：支持张开手、握拳、捏合等手势控制圣诞树的效果。
- **粒子效果**：通过 Three.js 创建粒子效果，表现出圣诞树、雪花、烟花等效果。
- **照片上传**：用户可以上传照片并将其挂到圣诞树上，装饰圣诞树。
- **手机兼容性**：支持在手机浏览器中使用手势操作和上传照片。

## 安装与使用

### 1. 克隆项目

首先，克隆本项目到本地：

```bash
git clone https://github.com/yourusername/christmas-tree-gesture.git
cd christmas-tree-gesture

2. 安装依赖

项目使用了 npm 管理依赖，运行以下命令来安装所有依赖项：
npm install

3. 本地开发

安装完成后，启动本地开发服务器：
npm start

浏览器会自动打开，并显示项目。您也可以手动在浏览器中访问 http://localhost:3000 进行测试。

4. 部署到 Vercel

将项目推送到 GitHub。

登录到 Vercel
 并点击 "New Project"。

选择 GitHub 仓库并按照 Vercel 的指示完成部署。

5. 项目目录结构
/christmas-tree-gesture
  ├── index.html      // 页面框架，包含手势和视频的显示

项目细节
1. 3D 圣诞树和粒子效果

使用 Three.js 创建的 3D 圣诞树，通过粒子系统表现树上的装饰、雪花、烟花等效果。

2. 手势识别

本项目使用 MediaPipe 的 HandLandmarker 模型进行手势识别。用户可以通过手势控制树的效果：

张开手：粒子炸开

握拳：粒子聚集形成圣诞树

捏合：触发照片上传和挂到树上

3. 照片上传与装饰

用户可以上传照片，并将其挂到圣诞树上。照片作为纹理贴图被附加到 3D 模型上，仿佛是树上的装饰。

4. 响应式设计

该项目已经适配了手机浏览器，可以通过触摸操作与项目进行交互。无论是在桌面设备还是手机上，都能获得流畅的体验。
